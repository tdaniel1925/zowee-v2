Component Props

TypeScript
Prop Type Default Description
variant 'default' | 'accent' | 'featured' 'default' Visual style — border, glow, and top-line treatment
padding 'sm' | 'md' | 'lg' 'md' Internal padding: sm=12px, md=20px, lg=28px
hoverable boolean true Enables translateY lift and border brightening on hover
glowPulse boolean true Enables animated glow loop (featured variant only)
className string undefined Additional CSS classes merged onto the card root element
onClick function undefined Makes card interactive; adds cursor-pointer and focus ring
children ReactNode required Card content — any valid React node
